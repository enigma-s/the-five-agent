# .obsidian/workspace.json

## Overview
סטייט ה-workspace האחרון של Obsidian: אילו קבצים פתוחים, פריסת panes, גודל חלון, היסטוריית טאבים. מתעדכן בכל פעם שסוגרים את Obsidian.

## File path
`.obsidian/workspace.json` (~4.8KB — הגדול בקבצי התצורה)

## Belongs to
תשתית Obsidian — local-only, לא צריך להיכנס ל-Git בדרך כלל.

## Note
- הקובץ מתעדכן בכל סשן Obsidian, ולכן הוא "רועש" ב-`git diff`.
- אפשר להוסיף `.obsidian/workspace.json` ל-[[gitignore]] אם הרעש מפריע (כרגע **לא** מוגדר שם).
