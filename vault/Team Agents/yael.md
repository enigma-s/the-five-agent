# יעל — כותבת התוכן

## Overview
חברת הצוות שאחראית על כל מה שקשור לטקסט: ניסוח, עריכה, יצירת תוכן, שיפור סגנון. ראובן ינתב אליה כל בקשה שמתמקדת בתוצר טקסטואלי.

## Defined in
מוזכרת ב-[[claude-md]] **ומוגדרת בקובץ סוכן** ב-`.claude/agents/yael.md` (מאז 2026-05-13).

## Role
- **תחום:** כתיבה ותוכן — שכתוב מאמרי גלם, עריכה, ניסוח מחדש, תרגום, סיכום
- **קלט:** מאמרי גלם בתיקיית `Content/`
- **תוצר:** שני קבצים לכל מאמר ב-`Output/` — Markdown נקי + HTML semantic מינימלי
- **כלים מאופשרים:** Read, Write, Edit, Glob, Grep בלבד
- **כלים אסורים:** Bash, WebSearch, API חיצוני, סוכנים אחרים
- **סגנון:** נטען מ-`yael/style-guide.md` ומ-`yael/reference/` בתחילת כל סשן (graceful אם חסרים)
- **לא מבצעת:** פרסום, מחקר, עיצוב ויזואלי
- **תלויות סביבה:** `ANTHROPIC_API_KEY` ב-[[env]]

## Status
**Operational** — קובץ הסוכן קיים, ראובן יכול לנתב אליה משימות. ה-style guide והדוגמאות (`yael/style-guide.md`, `yael/reference/`) עדיין לא נכתבו ע"י המשתמש — הסוכן מטפל בכך graceful אך כשייווצרו, איכות הפלט תשתפר.
