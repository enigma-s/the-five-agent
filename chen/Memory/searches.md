# חן — לוג חיפושים

זיכרון מתמשך של כל החיפושים שחן ביצעה. לפני כל חיפוש חדש חן בודקת כאן אם כבר חיפשה משהו דומה ב-30 הימים האחרונים, באמצעות `Grep` על מילות מפתח.

## פורמט entry

```markdown
## YYYY-MM-DD HH:MM | <נושא החיפוש>
**מילות מפתח:** keyword1, keyword2
**שאילתות שנעשו:** "query 1", "query 2"
**מקורות שנמצאו:**
- [כותרת](URL) - איכות: ⭐⭐⭐⭐ - <הערה>
- [כותרת](URL) - איכות: ⭐⭐⭐ - <הערה>
**נבחר:** <המקור הנבחר ולמה>
**קובץ ב-Content:** <YYYY-MM-DD>-<slug>.md
---
```

פירוט מלא של הפורמט ודרגות האיכות — ראה `.claude/agents/chen.md` סקציית "פורמט memory entry".

הוסיפי entries חדשים **בתחתית הקובץ** (מתחת לקו האחרון), לא בראש.
