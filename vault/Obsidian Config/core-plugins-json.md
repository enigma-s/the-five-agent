# .obsidian/core-plugins.json

## Overview
רשימת ה-core plugins המובנים של Obsidian, עם flag `true`/`false` לכל אחד. הקובץ הזה שולט באיזה יכולות זמינות (file explorer, graph view, daily notes, ועוד).

## File path
`.obsidian/core-plugins.json`

## Active (`true`)
| Plugin | תפקיד |
|---|---|
| `file-explorer` | צד שמאל — עץ קבצים |
| `global-search` | חיפוש כללי ב-vault |
| `switcher` | קפיצה בין קבצים (Ctrl+O) |
| `graph` | תצוגת גרף קשרים |
| `backlink` | תצוגת backlinks |
| `canvas` | קנבס לחזותית |
| `outgoing-link` | תצוגת outgoing links |
| `tag-pane` | פאנל tags |
| `properties` | נגישה לתכונות frontmatter |
| `page-preview` | תצוגת preview ב-hover |
| `daily-notes` | יומנים יומיים |
| `templates` | תבניות |
| `note-composer` | merge/split notes |
| `command-palette` | Ctrl+P |
| `editor-status` | סטטוס בעורך |
| `bookmarks` | מועדפים |
| `outline` | outline view |
| `word-count` | ספירת מילים |
| `file-recovery` | התאוששות קבצים |
| `sync` | סנכרון |
| `bases` | **קבצי `.base`** — תצוגות database-like |

## Inactive (`false`)
`footnotes`, `slash-command`, `markdown-importer`, `zk-prefixer`, `random-note`, `slides`, `audio-recorder`, `workspaces`, `publish`, `webviewer`

## Notable
`bases: true` — מאפשר שימוש ב-[[obsidian-bases]] אם נחליט להוסיף views.

## Belongs to
תשתית Obsidian.
