# Obsidian Config — Index

קבצי תצורה של אפליקציית Obsidian, יושבים תחת `.obsidian/` בשורש הריפו. נוצרו אוטומטית כש-Obsidian פתחה את התיקייה בפעם הראשונה. שולטים על איזה plugins פעילים, איך ה-workspace נראה, ותצורת תצוגה.

## Topics

- [[app-json]] — `.obsidian/app.json` — הגדרות בסיסיות של האפליקציה (כרגע `{}`)
- [[appearance-json]] — `.obsidian/appearance.json` — נושא ועיצוב (כרגע `{}`)
- [[core-plugins-json]] — `.obsidian/core-plugins.json` — איזה core plugins פעילים
- [[workspace-json]] — `.obsidian/workspace.json` — סטייט החלונות והטאבים האחרון
